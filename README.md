# [Group1] 台北市房價與租金預測分析
本專案旨在分析台北市房價與租金數據，並建立預測模型。專案包含數據收集、地理編碼處理、捷運站特徵提取等功能。主要目標是研究捷運站距離對房價和租金的影響，並建立有效的預測模型。
